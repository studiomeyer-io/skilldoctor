# Example Project — Agent Instructions

This is an example `AGENTS.md`. It is plain Markdown with no required frontmatter;
skilldoctor only runs content and security checks against it.

## Build and test

- Install dependencies: `npm install`
- Run the test suite: `npm test`
- Type-check: `npm run typecheck`

## Code style

- TypeScript strict mode. Avoid `any`.
- Prefer small, pure functions and clear names.

## Pull requests

- Keep PRs focused. Describe what changed and why.
- Make sure CI is green before requesting review.
